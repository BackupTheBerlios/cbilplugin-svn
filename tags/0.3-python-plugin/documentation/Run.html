How to Run a Python Script

Three methods
  1. Choose Run from the Python menu, this will prompt you with a file dialog
  2. Right click in the editor pane of an open python source file and select "Python Run"
  3. Right click on a python source in the project tree and select "Python Run"
Be aware that
  • If you have unsaved changes beware. The plugin will not automatically save your file(s) (this will probably change)
  • Currently, console output cannot be rerouted to a C::B window (this will change)
  • You can run multiple python sources simultaneously
  • The console will close upon program exit, so you might want to insert a pause into your python program with raw_input() (I may also fix this eventually)
