Python Toolbar

By default the toolbar is not shown in the workspace, but you can activate it from the view menu (PyPlugin). I've "borrowed" the gdbdebugger icons, so this should be familiar to regular codeblocks users (I'll change this eventually to distinguish it). Currently the plugin only responds to 4 buttons:
  • Debug Start/Continue (combined into one button)
  • Next
  • Step into
  • Stop

    See Debug for an explanation of these actions.
